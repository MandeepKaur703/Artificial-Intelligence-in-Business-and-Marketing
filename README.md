-AI in Marketing using Machine Learning

-Project Overview:
This project explores the role of Artificial Intelligence (AI) in modern marketing by combining research analysis with practical machine learning implementation. It focuses on comparing AI-driven, human-led, and hybrid marketing strategies to evaluate their effectiveness in terms of efficiency, engagement, conversion, and creativity.

---

-Objectives:
* To analyze the impact of AI in marketing decision-making
* To compare AI-based, human-based, and hybrid marketing approaches
* To implement machine learning models for customer purchase prediction
* To evaluate performance using accuracy and marketing metrics

---

-Dataset:
Online Shoppers Purchasing Intention Dataset
Contains user behavior data such as:
  * Page visits
  * Bounce rate
  * Exit rate
  * Visitor type
  * Product-related interactions
Target Variable: **Revenue (Purchase: Yes/No)**

---

Technologies Used:
* Python
* Google Colab
* Pandas, NumPy
* Scikit-learn
* Matplotlib

---

Machine Learning Models Implemented:
* Logistic Regression
* Support Vector Machine (SVM)
* Decision Tree
* Random Forest ⭐ (Best Performing Model)

---

Methodology
1. Data preprocessing (handling categorical variables, scaling)
2. Splitting dataset into training and testing sets
3. Training multiple machine learning models
4. Evaluating models based on accuracy
5. Comparing marketing strategies using synthesized dataset
6. Visualizing results (engagement, conversion, creativity)

---

Results & Findings:
* Random Forest achieved the highest accuracy (~89%)
* AI-driven marketing showed high efficiency and scalability
* Human-led marketing performed better in creativity and emotional connection
* Hybrid marketing approach delivered the best overall performance

---

Visualizations:
* Engagement Rate Comparison
* Conversion Rate Comparison
* Creativity Score Analysis
* Feature Importance Graph

---

Key Insights:
* AI enhances efficiency but lacks creativity
* Human input is essential for emotional engagement
* Hybrid approach is most effective in real-world marketing

---

* No primary data collection
